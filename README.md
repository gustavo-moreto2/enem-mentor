# ENEM Mentor

Site de estudos para o ENEM com foco em redação, português e treino guiado por IA.

## Site publicado

https://enem-mentor.vercel.app

## Rodar localmente

```powershell
npm start
```

Abra `http://localhost:5173`.

## Variável de ambiente

Configure a chave da Gemini apenas no ambiente:

```powershell
setx GEMINI_API_KEY "SUA_CHAVE"
```

Nunca coloque a chave em arquivos do projeto.

Feito para estudar ENEM.
